# 버인협회 🌸

API 금액 지표 대시보드 & 크루원 공약 관리 시스템

## 📋 기능

- ✨ 실시간 금액 추적
- 👑 수장 및 크루원 관리
- 📢 공지사항 기능
- 🔐 관리자 인증 시스템
- 💾 localStorage 기반 데이터 저장
- 📱 모바일 반응형 디자인

## 🎯 주요 기능

### 공개 대시보드
- 전체 누적 금액 및 멤버 수 표시
- 금액 분포 시각화 (막대 차트)
- 각 멤버별 공약 확인

### 관리자 기능
1. **공지사항** - 공지 추가/삭제
2. **금액 수정** - 멤버별 금액 편집
3. **멤버 관리** - 멤버 추가/삭제, 공약 관리
4. **초기화** - 데이터 초기화 옵션
